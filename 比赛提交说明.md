## 三、作品简介

开发者留存雷达是一款面向高校训练营运营场景的 Web 应用。系统通过二维码统一承接活动报名、账号开通、任务签到和最终作品提交，并在管理后台实时展示开发者留存漏斗、阶段转化率、参与者名单和作品链接，帮助运营人员快速定位开发者流失节点，持续提升训练营参与度和最终完成率。

本作品对应命题 B“促进开发者活跃”，通过可量化的运营数据帮助校园开发者持续参与华为云生态中的学习、探索与贡献。

## 四、核心功能

- 训练营及动态任务阶段管理，支持新增、编辑、删除和上下排序。
- 每个阶段自动生成独立二维码和学生免登录签到入口。
- 使用学号或手机号完成身份匹配，重复签到不会重复计数。
- 活动关闭后禁止公开签到，同时保留历史数据和审计记录。
- 最终作品链接校验、留档、后台查看和 CSV 导出。
- ECharts 实时漏斗、阶段转化率、整体完成率和最低转化节点提示。
- 支持日期范围筛选、参与者搜索、阶段筛选和漏斗图 PNG 导出。
- 提供 `100 → 80 → 50 → 35` 演示数据一键重置功能。
- 使用 Docker Compose、Node.js、React 和 PostgreSQL 部署至华为云 Flexus 应用服务器 L 实例。

## 六、演示方法

1. 打开管理后台并使用登录页展示的评委演示账号登录。
2. 在“数据概览”中展示实时漏斗、阶段转化率、时间筛选和图表导出。
3. 在“签到入口”中展示各阶段二维码和复制链接功能。
4. 扫描“活动报名”二维码填写信息，再返回后台刷新数据。
5. 在“参与者”页面展示搜索、阶段筛选、作品链接和 CSV 导出。
6. 在“活动设置”中展示阶段编辑、排序、活动状态切换和演示数据重置。
